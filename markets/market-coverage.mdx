## How to use this page

- Use it to understand which sources currently appear in Polaris catalog.
- Use it to see the current market count for each source.
- Use the separate tables to distinguish markets that are currently open from those that are still in preview.
- Use the example market IDs as a starting point only. Always confirm the exact `source` and `market` pair in [Catalog](/reference/catalog) before you build requests.
- Use [Catalog](/reference/catalog) to confirm exact `source` and `market` pairs.
- Use the [API overview](/reference/api-reference) if you already know the market but not the right delivery surface.
