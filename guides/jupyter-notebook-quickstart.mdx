If you are looking for the Hyperliquid "SpaceX" market, use `SPX` as the Polaris asset ID. Use [Catalog](/reference/catalog) to confirm the exact ID before you query data.

## Discover the exact market ID

Do not guess the asset name. Start with an exchange, list the returned asset IDs, filter by a likely ticker-style substring, then confirm the exact Polaris ID from the returned matches.

```python
from polaris_data import PolarisClient

exchange = "hyperliquid"
search = "SPX"  # Replace this with the substring you want to inspect.

with PolarisClient() as client:
    catalog = client.catalog(exchange=exchange)
    assets = [asset["id"] for asset in catalog["exchanges"][0]["assets"]]

print(assets[:20])

matches = [asset for asset in assets if search.upper() in asset.upper()]
print(matches)
```

Use one of the exact IDs returned in `matches` for later queries. If `matches` is empty, inspect the full `assets` list instead of guessing. On Hyperliquid, the market some users describe as "SpaceX" is exposed as `SPX`.

## Fetch recent public trades first

Use `trades(...)` with no API key and no `from_` or `to` when the recent public window is enough.

```python
import matplotlib.pyplot as plt
import pandas as pd
from polaris_data import PolarisClient

with PolarisClient() as client:
    trades = client.trades(exchange="hyperliquid", asset="SPX")

if not trades:
    raise ValueError(
        "No trades returned. Confirm the exact Polaris asset ID, check the recent public window, or try the authenticated ranged query below."
    )

print(trades[0])

df = pd.DataFrame(trades)
df["time"] = pd.to_datetime(df["timestamp"], unit="us", utc=True)
df["price"] = df["data"].map(lambda row: row["price"])
df["quantity"] = df["data"].map(lambda row: row["quantity"])
df["side"] = df["data"].map(lambda row: row["side"])

display(df.head())
```

```python
fig, ax = plt.subplots(figsize=(12, 5))

buy_trades = df[df["side"] == "buy"]
sell_trades = df[df["side"] == "sell"]

ax.scatter(
    buy_trades["time"],
    buy_trades["price"],
    s=buy_trades["quantity"] * 8,
    alpha=0.5,
    label="Buy",
)
ax.scatter(
    sell_trades["time"],
    sell_trades["price"],
    s=sell_trades["quantity"] * 8,
    alpha=0.5,
    label="Sell",
)

ax.set_title("Hyperliquid SPX trades")
ax.set_xlabel("Time (UTC)")
ax.set_ylabel("Price")
ax.legend()
plt.xticks(rotation=30)
plt.tight_layout()
plt.show()
```

This gives you a simple trade plot where marker size reflects executed quantity.

## Use an API key for explicit time ranges

Add `POLARIS_API_KEY` when you want an explicit time range instead of the recent public window.

```python
import os
from datetime import datetime, timedelta, timezone

from polaris_data import PolarisClient

os.environ["POLARIS_API_KEY"] = "pk_live_your_key"

end = datetime.now(timezone.utc)
start = end - timedelta(hours=2)

with PolarisClient() as client:
    trades = client.trades(
        exchange="hyperliquid",
        asset="SPX",
        from_=start,
        to=end,
    )

if not trades:
    raise ValueError(
        "No trades returned for this range. Move the time window, confirm the asset ID, or verify that your access policy covers the requested data."
    )

print(trades[0])
```

After this cell, rerun the DataFrame and plotting cells above to analyze the authenticated ranged result set.

## Use OHLCV when you want a fast price chart

Use `ohlcv(...)` when you want a quick candle or line chart instead of individual trades.

```python
import matplotlib.pyplot as plt
import pandas as pd
from polaris_data import PolarisClient

with PolarisClient() as client:
    bars = client.ohlcv(
        exchange="hyperliquid",
        asset="SPX",
        interval="1m",
    )

if not bars:
    raise ValueError(
        "No OHLCV bars returned. Confirm the asset ID, check the recent public window, or switch to an authenticated ranged query."
    )

print(bars[0])

bars_df = pd.DataFrame(bars)
bars_df["time"] = pd.to_datetime(bars_df["timestamp"], unit="us", utc=True)

bars_df.plot(x="time", y="close", figsize=(12, 5), legend=False, title="Hyperliquid SPX close")
plt.xlabel("Time (UTC)")
plt.ylabel("Close")
plt.tight_layout()
plt.show()
```

Use this path when you need a quick market view. Switch back to `trades(...)` when you need execution detail.

## What the SDK returns

- `catalog(...)` returns a JSON object with exchanges and exact asset IDs.
- `trades(...)` returns a list of normalized trade events.
- `ohlcv(...)` returns a list of bars. Each bar includes `timestamp`, `open`, `high`, `low`, `close`, `volume`, `trades`, and `interval`.
- `timestamp` values are UTC microseconds. Convert them with `pd.to_datetime(..., unit="us", utc=True)`.

## Troubleshooting

<Tip>
  If you see `No trades returned` or `No OHLCV bars returned`, check the exact Polaris asset ID from catalog, remember that the no-key path only uses the recent public window, confirm that your access policy covers the data you want, and verify that your requested time range is not empty.
</Tip>
