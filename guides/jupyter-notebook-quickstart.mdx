Use this guide when you want a notebook that confirms a market ID, loads data into a DataFrame, and plots it quickly.

This walkthrough stays focused on fetching `BTC` data from `hyperliquid`. If you want a different market or venue, start with [Market Coverage](/markets/market-coverage) and then confirm the exact ID in [Catalog](/reference/catalog) before you query data.

## Discover the exact market ID

Do not guess the market name. Ask catalog for the exact source and market pair you want to query, then inspect the returned market row before you fetch data.

```python
import pandas as pd
from polaris_data import PolarisClient

source = "hyperliquid"
market = "BTC"

with PolarisClient() as client:
    catalog_result = client.catalog(source=source, market=market)

source_row = catalog_result["sources"][0]
markets_df = pd.DataFrame(source_row["markets"])

display(markets_df[["id", "start", "end"]])
```

Use the exact `id` returned by catalog for later queries. If you want a different market or venue, browse [Market Coverage](/markets/market-coverage) first and then query [Catalog](/reference/catalog) with the exact `source` and `market` pair you want.

## Fetch recent public OHLCV first

Use `ohlcv(...)` with no API key and no `from_` or `to` when you want the most reliable first chart from the recent public window.

```python
import matplotlib.pyplot as plt
import pandas as pd
from polaris_data import PolarisClient

with PolarisClient() as client:
    bars = client.ohlcv(
        source="hyperliquid",
        market="BTC",
        interval="1m",
    )

if not bars:
    raise ValueError(
        "No OHLCV bars returned. Confirm the exact Polaris market ID, check the recent public window, or try the authenticated ranged query below."
    )

print(bars[0])

bars_df = pd.DataFrame(bars)
bars_df["time"] = pd.to_datetime(bars_df["timestamp"], unit="us", utc=True)

display(bars_df.head())
```

```python
bars_df.plot(
    x="time",
    y="close",
    figsize=(12, 5),
    legend=False,
    title="Hyperliquid BTC close",
)
plt.xlabel("Time (UTC)")
plt.ylabel("Close")
plt.tight_layout()
plt.show()
```

This gives you a fast first chart with a liquid default market and a simple notebook-friendly bar shape.

## Use trades when you need execution detail

Use `trades(...)` after the bar chart when you want execution-level detail instead of interval bars.

```python
import matplotlib.pyplot as plt
import pandas as pd
from polaris_data import PolarisClient

with PolarisClient() as client:
    trades = client.trades(
        source="hyperliquid",
        market="BTC",
    )

if not trades:
    raise ValueError(
        "No trades returned. Confirm the exact Polaris market ID, check the recent public window, or switch back to OHLCV for a faster market view."
    )

print(trades[0])

trades_df = pd.DataFrame(trades)
trades_df["time"] = pd.to_datetime(trades_df["timestamp"], unit="us", utc=True)
trades_df["price"] = trades_df["data"].map(lambda row: row["price"])
trades_df["quantity"] = trades_df["data"].map(lambda row: row["quantity"])
trades_df["side"] = trades_df["data"].map(lambda row: row["side"])

display(trades_df.head())
```

```python
fig, ax = plt.subplots(figsize=(12, 5))

buy_trades = trades_df[trades_df["side"] == "buy"]
sell_trades = trades_df[trades_df["side"] == "sell"]

ax.scatter(
    buy_trades["time"],
    buy_trades["price"],
    s=buy_trades["quantity"] * 8,
    alpha=0.5,
    label="Buy",
)
ax.scatter(
    sell_trades["time"],
    sell_trades["price"],
    s=sell_trades["quantity"] * 8,
    alpha=0.5,
    label="Sell",
)

ax.set_title("Hyperliquid BTC trades")
ax.set_xlabel("Time (UTC)")
ax.set_ylabel("Price")
ax.legend()
plt.xticks(rotation=30)
plt.tight_layout()
plt.show()
```

Use this path when you need execution detail. Stay with `ohlcv(...)` when you only need a quick market view.

## Use an API key for explicit time ranges

Add `POLARIS_API_KEY` when you want the notebook to run an explicit ranged query instead of relying on the recent public window.

```python
import os
from datetime import datetime, timedelta, timezone

from polaris_data import PolarisClient

os.environ["POLARIS_API_KEY"] = "pk_live_your_key"

end = datetime.now(timezone.utc)
start = end - timedelta(hours=6)

with PolarisClient() as client:
    bars = client.ohlcv(
        source="hyperliquid",
        market="BTC",
        from_=start,
        to=end,
        interval="1m",
    )

print(bars[0])
```

After this cell, rerun the OHLCV DataFrame and plotting cells above to analyze the ranged result set. Apply the same `from_` and `to` pattern to `trades(...)` when you need execution-level detail for a specific window.

## What the SDK returns

- `catalog(...)` returns a JSON object with sources and exact market IDs. See [Catalog](/reference/catalog) for the response shape.
- `ohlcv(...)` returns a list of bars. Each bar includes `timestamp`, `open`, `high`, `low`, `close`, `volume`, `trades`, and `interval`. See [OHLCV](/sdks/ohlcv).
- `trades(...)` returns a list of normalized trade events. See [Trades](/sdks/trades).
- `timestamp` values are UTC microseconds. Convert them with `pd.to_datetime(..., unit="us", utc=True)`.

## Troubleshooting

<Tip>
  If you see `No OHLCV bars returned` or `No trades returned`, check the exact Polaris market ID from catalog, remember that the no-key path only uses the recent public window, confirm that your access policy covers the data you want, and verify that your requested time range is not empty.
</Tip>
