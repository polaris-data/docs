Polaris provides REST endpoints at `https://api.polaris.supply`. Most reference endpoints are public. Add bearer auth only when you need broader historical access, account-tier visibility, or a dataset policy that requires it. Choose the endpoint family that matches your task, then read the smaller page for that endpoint.

## Endpoint families

### Metadata

- `GET /catalog` for public market discovery, with optional account-tier visibility when you send bearer auth

### Historical files

- `GET /snapshots` for public snapshot listing
- `GET /snapshots/download` for public or policy-gated file download, depending on the dataset

### Recent query endpoints

- `GET /trades` for public recent trades, with bearer auth for explicit historical bounds
- `GET /events` for public recent event streams, with bearer auth for explicit historical bounds
- `GET /ohlcv` for public recent bars, with bearer auth for explicit historical bounds
- `GET /raw`

## Common response patterns

`/trades`, `/events`, and `/raw` return paginated JSON responses:

```json
{
  "data": [],
  "next_cursor": "opaque_token_or_null",
  "has_more": false
}
```

`/ohlcv` returns a simpler JSON envelope with `format`, `data`, `next_cursor`, and `has_more`, or a TradingView-shaped payload when you request `format=tradingview`.
