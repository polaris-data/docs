---
title: "Rust SDK"
description: "Install the Polaris Rust client and authenticate with your existing API key."
---

The Polaris Rust SDK is available on crates.io as [`polaris-data`](https://crates.io/crates/polaris-data).

## Install

```bash
cargo add polaris-data
```

## Authentication

Use the same Polaris API key you use for REST requests.

```bash
export POLARIS_API_KEY="pk_live_your_key"
```

See [Connecting](/guides/connect) for the shared auth model.
