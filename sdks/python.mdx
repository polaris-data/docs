---

The Polaris Python SDK is available on PyPI as [`polaris-data`](https://pypi.org/project/polaris-data/).

## Install

```bash
pip install polaris-data
```

## Authentication

Use the same Polaris API key you use for REST requests.

```bash
export POLARIS_API_KEY="pk_live_your_key"
```

See [Connecting](/guides/connect) for the shared auth model.
